# vue-cypress-example

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Run your cypress tests

```
yarn cypress open-ct
```

### Lints and fixes files

```
yarn lint
```
